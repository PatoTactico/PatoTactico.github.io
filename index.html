Proyecto MyTablero

En este proyecto se encuentran incluidos muchos aspectos como los son los formularios, las bases de datos, no podría faltar su diseño, estructura y funcionamiento. A continuación, unas breves definiciones sobre formularios, bases de datos y con que tipos de lenguajes realizaremos su diseño, estructura y funcionamiento.


El objetivo de la creación de esta página web es el de que los padres de familia puedan tener un control, así como una forma más accesible para supervisar si sus hijos han hecho y entregado sus tareas en tiempo y forma. Además de que los mismos alumnos tengan un apoyo para poder saber que tareas tienen, cuales aún no han entregado y cuales ya han sido entregadas, así como saber cuál es su calificación de dicha tarea.


Diseño

Diseño El diseño es un aspecto visual primordial para que una página web sea atractiva y el público pueda disfrutar al navegar por esta misma. El lenguaje que se utiliza para darle diseño a una página web es CSS (en inglés Cascading Style Sheets); es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado como lo es HTML. Así, a los elementos de la página web creados con HTML se les dará la apariencia que se desee utilizando CSS: colores, espacios entre elementos, tipos de letra, entre otros.


Estructura

La estructura es el aspecto principal y fundamental para el inicio de una página web, se puede decir que es el esqueleto de nuestra página. Se compone de tres partes: una línea que contiene información sobre la versión de HTML, una sección de cabecera declarativa (delimitada por el elemento HEAD), un cuerpo, que contiene el contenido real del documento.

Funcionamiento

El funcionamiento es la parte intermedia al elaborar una página web, es el nivel en donde hacemos que cualquier elemento de la página tenga su funcionamiento dependiendo de lo que se indique con las funciones y variantes. El lenguaje que se utiliza para el funcionamiento es JavaScript, es capaz de detectar errores en formularios, de crear bonitos sliders que se adapten a cualquier pantalla, de hacer cálculos matemáticos de forma eficiente, de modificar elementos de una página web de forma sencilla.
